# AIoT for Smart Hotels

AIoT-based smart hotel system developed as a graduation project.  
The project combines IoT sensors and intelligent services to monitor hotel environments and optimize energy consumption.

The objective is to build a **smart hotel infrastructure** capable of collecting environmental data, analyzing it and triggering automated services.

## Project Overview

This project explores the integration of **Artificial Intelligence and Internet of Things (AIoT)** in hotel infrastructures.

The system allows:

- Monitoring environmental conditions in hotel rooms
- Collecting sensor data
- Processing information through intelligent services
- Improving energy consumption efficiency
- Automating certain hotel operations

---

## System Workflow

1. IoT sensors collect environmental data from hotel rooms  
2. The collected data is transmitted to the system  
3. The platform processes the data  
4. Intelligent services analyze the conditions  
5. Automated actions or monitoring insights are generated
